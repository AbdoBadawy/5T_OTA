# ITI_lab_5T_OTA
Desing of 5T OTA using ADT and simulated on Xschem and Ngspice in 180n technology.
## Specifications:
Technology: gf180mcu

load capacitance = 5p

Open Loop DC gain >= 34dB

CMRR >= 74dB

PM >= 70

CM input Low <= 1V

CM input High >= 1.5V

GBW >= 10MHz

## Achieved:

Open Loop DC gain = 50.2

CMRR = 86dB

PM = 89

CM input Low = 0.82V

CM input High = 1.65V

GBW = 10.3MHz
